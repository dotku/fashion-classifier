# Task

1. Image Upload + AI Classification

[x] Users can upload garment photos through the web app.

[x] Each image should be processed by a multimodal AI model that returns both a rich natural-language description and structured attributes such as garment type, style, material, color palette, pattern, season, occasion, consumer profile, trend notes, and location context.

[x] Store both the descriptive output and the structured metadata.

> Both the descriptive output and structured metadata are saved to a SQLite database at data/fashion.db:

1. Search + Filtering

[x] Display the image library in a visual grid.

[x] Support filtering across garment attributes such as garment type, style, material, color palette, pattern, occasion, consumer profile, and trend notes.

[x] Support contextual filters including location (continent, country, city), time (year, month, season), and designer.

[x] Filters should be dynamically generated from the data rather than hardcoded.

[x] Support full-text search across image descriptions using natural queries such as "embroidered neckline" or "artisan market."

1. Designer Annotations

[x] Allow users to add their own tags, notes, and observations in addition to AI-generated metadata. • Annotations should be searchable and clearly distinguished from AI-generated output.
